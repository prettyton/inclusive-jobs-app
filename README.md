# Inclusive Jobs

An accessible job platform designed to connect people with disabilities to inclusive employment opportunities.

## Features

- Multi-disability type selection
- Accessible UI design
- Tailored job recommendations based on accessibility needs
- Clean, modern interface

## Disability Categories Supported

- Visual Impairment
- Hearing Impairment
- Mobility
- Cognitive / Learning
- Neurodivergent
- Mental Health
- Speech

## Tech Stack

- React.js
- Lucide Icons
- CSS-in-JS styling

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/prettyton/inclusive-jobs-app.git
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
